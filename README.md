# Лабораторная работа 2

## Overview
Главный репозиторий для лабораторной работы №2. Содержит подмодули для заданий 1, 2, 3.

## Author
Муравейко Игнат, группа 13

## Usage
Каждый подмодуль собирается отдельно. См. раздел Modules.

## Modules
- [tpmp-lab2-task3](tpmp-lab2-task3) – задание 3 (структуры).  
  Сборка: `cd tpmp-lab2-task3 && make`  
  Запуск: `./tpmp-lab2-task3/bin/abon`
- [tpmp-lab2-task4](tpmp-lab2-task4) – задание 4 (работа с файлами).  
  Сборка: `cd tpmp-lab2-task4 && make`  
  Запуск: `./tpmp-lab2-task4/bin/trip`
